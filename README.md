# EasyEffects Preset

A custom audio preset for EasyEffects.

## 🎧 What this EQ does
In short, this preset makes your sound:
* **Deep** (Boosts super low bass)
* **Clear** (Cleans up muddy mid-tones)
* **Crisp** (Brings out vocals and high notes)

## 🚀 How to Install

### 1. Place the file
Save your JSON file into this folder:
```bash
~/.config/easyeffects/output/
```
*(You can name it anything, like `my_preset.json`)*

### 2. Load the preset
1. Open **EasyEffects**.
2. Click **Presets** on the top left.
3. Select your file and click **Load**.
